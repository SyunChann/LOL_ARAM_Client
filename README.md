# 아수라장 전적

롤 클라이언트의 로컬 LCU API를 읽어 `무작위 총력전: 아수라장` 전적을 보여주는 Windows 데스크톱 앱입니다.

## 주요 기능

- 롤 클라이언트의 최근 `KIWI` 경기만 자동 조회
- 승리(파랑)·패배(빨강) 구분, KDA, 피해량, 골드, 난전 지수 표시
- 경기 목록에서 챔피언, 스펠, 아이템, 증강 및 다른 플레이어의 증강 미리보기
- 브론즈·실버·골드·프리즘 증강 희귀도별 아이콘 색상 구분
- 아이템·스펠·증강 아이콘에 마우스를 올리면 이름과 설명 팝업 표시
- 클릭 한 번으로 팀별 상세 스코어보드, 아이템, 증강, 피해량 비교 확인
- 챔피언별 승률과 평균 딜량 요약

## 사용 전 준비

- Windows에서 롤 클라이언트를 실행하고 로그인해야 합니다.
- 앱은 현재 PC의 롤 클라이언트 `lockfile`과 로컬 LCU API만 사용합니다.
- LCU 전적 조회에 Riot API 키, Node.js, 별도 웹 서버는 필요하지 않습니다.

## Windows 설치 및 실행

배포된 파일 중 하나를 사용합니다.

- `아수라장 전적 Setup x.y.z.exe`: 일반 설치형입니다. 다른 PC에는 이 파일만 전달하면 됩니다.
- `아수라장 전적 x.y.z.exe`: 설치 없이 실행하는 포터블 버전입니다.

롤 클라이언트를 로그인 상태로 둔 뒤 앱을 실행하고 `새로고침`을 누릅니다.

## 개발 실행

```powershell
npm install
npm run dev -- --port 5173
```

브라우저에서 `http://localhost:5173`으로 접속합니다.

데스크톱 앱을 로컬에서 실행하려면 다음을 사용합니다.

```powershell
npm run desktop
```

## Windows 패키징

```powershell
npm run package:win
```

설치형과 포터블 실행 파일은 `release` 폴더에 생성됩니다. 빌드 산출 웹 파일은 `web-dist`에 생성됩니다.

## 참고

- 증강 정보는 클라이언트가 경기 데이터에 제공한 경우에만 보입니다.
- Riot 클라이언트의 로컬 API는 공식적으로 안정성을 보장하는 공개 API가 아니므로, 클라이언트 업데이트에 따라 일부 필드가 달라질 수 있습니다.
- 기존 공식 Riot API 실험 경로를 사용할 때만 `.env`에 `RIOT_API_KEY`를 설정합니다.
